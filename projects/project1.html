DC Dental | Website Development

Concept & Context

DC Dental is a dental clinic based in Brussels, Belgium. This project focused on creating a professional website to present the company, its services, and facilitate client appointment booking. The goal was to provide a clear and modern web presence aligned with the clinic’s brand identity.

Tools & Stack

HTML CSS JavaScript GitHub Figma

My Journey in the Project

I was solely responsible for the entire website project—from gathering requirements to final deployment. I designed the prototype using Figma, held meetings with the client to refine the design, and built the site using HTML, CSS, and JavaScript. I also hosted it using GitHub Pages and structured the content for future integration with the clinic's appointment system.

Build & Integration

The project began with a client meeting to define the brand tone and visual direction. I created a high-fidelity Figma prototype and iterated on it based on client feedback. Once approved, I developed the site using semantic HTML, styled it with CSS, and added interactive elements with JavaScript. Hosting and version control were managed through GitHub.

Obstacles & Fixes

A major challenge was the future integration with the clinic’s appointment software, which relies on a closed-access system. To address this, I structured the site to allow for easy backend connection later and ensured the layout accommodates a booking flow once integration is available.

Final Output

The clinic was highly satisfied with the final result. The website is now live and actively used by clients to learn about services and contact the clinic. The foundation is ready for booking system integration as soon as access is provided.

Final Thoughts

This project sharpened my client communication, web design, and front-end development skills. It also gave me experience in working with real constraints, such as third-party system limitations, and preparing for future backend integration.
